# Public dashboard data

These files contain only aggregated, disclosure-checked results. Country-year pathogen–drug cells with fewer than 30 tested isolates are suppressed. The public dashboard does not load raw or isolate-level ATLAS data.

Phase 8 Parquet tables add endpoint coverage, annual One Health and R&D trends,
exploratory trajectories, lag estimates and data-availability summaries. They
remain aggregated and contain no isolate-level records.

Redistribution remains subject to confirmation under the applicable Vivli
data-use agreement. Passing the technical disclosure checks does not by itself
grant permission to publish derived data.

`SHA256SUMS` records the exact public bundle verified for release `v0.7.0`.
