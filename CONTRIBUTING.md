# Contributing

Changes must preserve the frozen scientific definitions in `config/` unless a
new protocol version is explicitly declared. Do not commit raw or isolate-level
Vivli data, credentials or unsuppressed small cells.

Before opening a pull request:

```bash
pip install -e ".[dev]"
python -m pytest -q
python scripts/validate_release.py
ruff check src tests scripts/validate_release.py
```

Every change to a displayed value must be generated by the scripted pipeline.
Do not manually edit analytical result tables or plotted values.
